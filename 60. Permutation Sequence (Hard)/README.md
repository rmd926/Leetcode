# 60. Permutation Sequence (Hard)

## Problem Description
The set `[1, 2, 3, ..., n]` contains a total of `n!` unique permutations.

By listing all permutations in **lexicographic order** (dictionary order), we can label them from `1` to `n!`.

Given two integers `n` and `k`, return the **kth permutation sequence** of the numbers `1` to `n`.

---

## Examples

### Example 1
**Input:** `n = 3, k = 3`  
**Output:** `"213"`

### Example 2
**Input:** `n = 4, k = 9`  
**Output:** `"2314"`

### Example 3
**Input:** `n = 3, k = 1`  
**Output:** `"123"`

---

## Constraints
- `1 <= n <= 9`
- `1 <= k <= n!`
