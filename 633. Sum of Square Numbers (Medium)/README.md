# 633. Sum of Square Numbers (Medium)

## Problem Description
Given a non-negative integer `c`, determine whether there exist two integers `a` and `b` such that:

\[
a^2 + b^2 = c
\]

Return `true` if such integers exist, otherwise return `false`.

---

## Examples

### Example 1
**Input:** `c = 5`  
**Output:** `true`  

**Explanation:**  
`1^2 + 2^2 = 1 + 4 = 5`

### Example 2
**Input:** `c = 3`  
**Output:** `false`

---

## Constraints
- `0 <= c <= 2^{31} - 1`
