# 17. Letter Combinations of a Phone Number (Medium)

## Problem Description
Given a string `digits` containing digits from `'2'` to `'9'` inclusive, return all possible letter combinations that the number could represent, based on the classic phone keypad mapping. Return the answer in any order. The digit `'1'` does not map to any letters.

## Examples
### Example 1
**Input:** `digits = "23"`  
**Output:** `["ad","ae","af","bd","be","bf","cd","ce","cf"]`

### Example 2
**Input:** `digits = "2"`  
**Output:** `["a","b","c"]`

## Constraints
- `1 <= digits.length <= 4`
- `digits[i]` is a digit in the range `['2', '9']`
