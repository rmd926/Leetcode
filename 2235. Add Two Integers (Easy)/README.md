# 2235. Add Two Integers (Easy)

## Problem Description
Given two integers `num1` and `num2`, return the sum of the two integers.

---

## Examples

### Example 1
**Input:** `num1 = 12`, `num2 = 5`  
**Output:** `17`  
**Explanation:** `12 + 5 = 17`

### Example 2
**Input:** `num1 = -10`, `num2 = 4`  
**Output:** `-6`  
**Explanation:** `-10 + 4 = -6`

---

## Constraints
- `-100 <= num1, num2 <= 100`
