# 215. Kth Largest Element in an Array (Medium)

## Problem Description
You are given an integer array `nums` and an integer `k`. Return the **kth largest element** in the array.

Note:
- The answer is the **kth largest in sorted order**, **not** the kth **distinct** element.

The problem also asks: **Can you solve it without sorting?**

---

## Examples

### Example 1
**Input:** `nums = [3,2,1,5,6,4]`, `k = 2`  
**Output:** `5`

### Example 2
**Input:** `nums = [3,2,3,1,2,4,5,5,6]`, `k = 4`  
**Output:** `4`
