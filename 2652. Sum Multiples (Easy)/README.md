# 2652. Sum Multiples (Easy)

## Problem Description
Given a positive integer `n`, compute the sum of all integers in the range `[1, n]` (inclusive) that are divisible by **3**, **5**, or **7**.

Return the sum of all numbers in the given range that satisfy the condition.

---

## Examples

### Example 1
**Input:** `n = 7`  
**Output:** `21`

**Explanation:**  
Numbers in `[1, 7]` divisible by 3, 5, or 7 are `3, 5, 6, 7`.  
Sum = `3 + 5 + 6 + 7 = 21`.

### Example 2
**Input:** `n = 10`  
**Output:** `40`

**Explanation:**  
Numbers in `[1, 10]` divisible by 3, 5, or 7 are `3, 5, 6, 7, 9, 10`.  
Sum = `40`.

### Example 3
**Input:** `n = 9`  
**Output:** `30`

**Explanation:**  
Numbers in `[1, 9]` divisible by 3, 5, or 7 are `3, 5, 6, 7, 9`.  
Sum = `30`.

---

## Constraints
- `1 <= n <= 10^3`
