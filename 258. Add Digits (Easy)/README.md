# 258. Add Digits (Easy)

## Problem Description
Given a non-negative integer `num`, repeatedly add all of its digits until the result has only one digit, then return that single digit.

## Examples
### Example 1
**Input:** `num = 38`  
**Output:** `2`  
**Explanation:** `38 -> 3 + 8 = 11 -> 1 + 1 = 2`

### Example 2
**Input:** `num = 0`  
**Output:** `0`

## Constraints
- `0 <= num <= 2^31 - 1`

## Follow Up
Can you solve it without any loop or recursion in `O(1)` time?
